Customer Segmentation Analysis

This project focuses on segmenting customers based on their age, income, and spending behavior using machine learning techniques. The main objective is to identify patterns in customer data and group them into meaningful segments for better understanding.

The dataset used is the Mall Customer Segmentation Dataset from Kaggle, which includes features like Customer ID, Gender, Age, Annual Income, and Spending Score. Basic data preprocessing was performed, including handling data, encoding categorical values, and scaling numerical features.

K-Means clustering was applied to group customers, and the Elbow Method was used to determine the optimal number of clusters. Through this approach, different types of customer segments were identified, such as:

💎 High income – high spending
💸 Low income – low spending
⚖️ Average customers

The project was implemented using Python with libraries like Pandas, NumPy, Scikit-learn, Matplotlib, and Seaborn. It demonstrates basic understanding of clustering, data preprocessing, and visualization techniques.
